# Development Workflow

How to execute coding tasks.

- [OODA loop](../dreams/OODA.md) — every non-trivial step begins with observation
- [Field General](../dreams/FIELD-GENERAL.md) — operating framework for decisions
- [Git workflow](GIT-WORKFLOW.md) — commits, worktrees, stacks

## Rules

- [**Every non-trivial step begins with observation**][qbt-001]
- Clarify the task in one sentence before starting
- Break work into the smallest code deliverable
- Verify after each change, not at the end
- Summarize with TL;DR: Observed, Decided, Next steps, Blockers.
- Highlight tradeoffs and any deferred work
