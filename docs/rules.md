# Rules

This file is language-agnostic. Apply these defaults in Ruby, Python, and other languages unless project-local rules override them.

- When developing, see `docs/dev/WORKFLOW.md`.
- When performing git operations (commits, worktrees, stacks, Graphite), always apply `docs/dev/GIT-WORKFLOW.md`.
- When making non-trivial decisions, always apply OODA loop from `docs/dreams/OODA.md`.
- When making strategic or operating decisions, apply `docs/dreams/FIELD-GENERAL.md`.
- When considering when to stop working, always apply `docs/dreams/STOP-TINKERING.md`. Apply a hard stop if the heuristic score exceeds 8.
